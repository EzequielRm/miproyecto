# Página oficial de AnimAl
Enlace de demostración: https://ezequielrm.github.io/miproyecto/

## Acerca del proyecto
AnimAl es un sitio web estático que muestra mascotas en adopción y facilita el contacto con refugios y personas interesadas en adoptar.

## Características
- Galería de mascotas en adopción y adoptadas.
- Navegación de páginas de servicios y contacto.
- Diseño responsive para móviles y tablets.
- Menú tipo hamburguesa para versión móvil.
- Animación de campos en formularios de contacto.

## Tecnologías
- HTML5
- CSS3
- normalize.css
- JavaScript (vanilla )
